Quant a WebProtégé

WebProtégé és un editor col·laboratiu d'ontologies OWL per a navegadors web.

WebProtégé està desenvolupat per l'equip Protégé al Grup de Recerca en Informàtica Biomèdica (BMIR) de la Universitat d'Stanford, Califòrnia, EUA.

El treball està recolzat pel Grant GM121724 de l'Institut Nacional de Ciències Mèdiques Generals (NIGMS) als Instituts Nacionals de Salut dels Estats Units.
